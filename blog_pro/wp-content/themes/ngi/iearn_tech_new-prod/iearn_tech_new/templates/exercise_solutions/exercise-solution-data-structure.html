Write a Python program to create an Enum object and display a member name and value.

Sample data :

Afghanistan = 93

Albania = 355

Algeria = 213

Andorra = 376

Angola = 244

Antarctica = 672

Expected Output :

Member name: Albania

Member value: 355

In [2]:
from enum import Enum
class Country(Enum):
    Afghanistan = 93
    Albania = 355
    Algeria = 213
    Andorra = 376
    Angola = 244
    Antarctica = 672
print('\nMember name: {}'.format(Country.Albania.name))
print('Member value: {}'.format(Country.Albania.value))

Member name: Albania
Member value: 355

Write a Python program to find the class wise roll number from a tuple-of-tuples.

Expected Output:

defaultdict(, {'VII': [1], 'V': [1, 2], 'VI': [1, 2, 3]})

In [8]:
from collections import defaultdict
classes = (
    ('V', 1),
    ('VI', 1),
    ('V', 2),
    ('VI', 2),
    ('VI', 3),
    ('VII', 1),
)

class_rollno = defaultdict(list)

for class_name, roll_id in classes:
    class_rollno[class_name].append(roll_id)

print(class_rollno)

defaultdict(<class 'list'>, {'V': [1, 2], 'VI': [1, 2, 3], 'VII': [1]})

Write a Python program to compare two unordered lists (not sets)

In [9]:
from collections import Counter
def compare_lists(x, y):
    return Counter(x) == Counter(y)
n1 = [20, 10, 30, 10, 20, 30]
n2 = [30, 20, 10, 30, 20, 50]
print(compare_lists(n1, n2))

False
