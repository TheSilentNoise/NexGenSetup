Project

Design and Develop a Price Comparison Application to compare price of courses from various websites. You should compare prices of the courses with below key words:

  1. Python
  2. Statistics
  3. Data Analysis
  4. Data Visualization

You should compare prices from the below websites:

  1. Coursera
  2. Udacity
  3. Edureka
  4. Simplilearn
  5. Iearn

The application should read name of courses and websites from an external configuration File. The application should have exception handling. The application should log messages in an external log file. The application should notify if a particular course is not present in any particular site. The application should notify which site provides the lowest and highest price for a particular course.
