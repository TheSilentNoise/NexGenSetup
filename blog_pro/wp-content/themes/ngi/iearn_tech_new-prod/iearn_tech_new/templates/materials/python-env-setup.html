Python Installation

Install Python in Mac

1.Open Terminal.

2.Install and set up Homebrew. Type the below in the terminal.

Homebrew provides OS X with a free and open source software package managing system that simplifies the installation of software on OS X.

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

3.Now, put Homebrew directory on top of PATH environment variable. This will enable to use brew commands from the terminal under any directory in the file system.

$ nano ~/.bash_profile

Write the following content

export PATH=/usr/local/bin:$PATH

Save the file and exit

$ source ~/.bash_profile

4.Now install Python3 or python2 depending upon your requirements

$ brew install python3

5.To install a particular package

$ pip3 install <packagename>
Eg. – pip3 install numpy

6.Create a Virtual Environment while coding in Python. Virtual environments enable you to have an isolated space on your computer for Python projects, ensuring that each of your projects can have its own set of dependencies that won’t disrupt any of your other projects.

$ mkdir PythonProject
$ cd PythonProject
$ python3 -m venv env #Creates a virtual environment
$ source env/bin/activate
(env) $ pip install scipy

Note, that the scipy package would be installed inside the particular environment (env) only, not in globally.

7.Create simple program in Python

(env) $ nano hello.py

print(“Hello, World!”)

Save and exit (env) $ python hello.py

Output :-

    Hello, World!

Install Python in Linux

Python2.7 is generally pre-installed in all Linux distributions

To check,

1.Open Terminal

$ python --version
Python 2.7.12

2.To install Python 3 versions,

On Debian derivatives such as Ubuntu, use APT:

$ sudo apt-get install python3

On Red Hat and derivatives, use yum:

$ sudo yum install python

On SUSE and derivatives, use zypper

$ sudo zypper install python3

3.To check Python 3 version:-

$ python3 –version

Install Python in Windows

1.Downloading

 - Download python from - here
 - Click the Download Python 3.6.x button.

2.Installing

- Double-click the icon labeling the file **python-3.6.x.exe**.
  An **Open File - Security** Warning pop-up window will appear.
- Click **Run**.
- A Python 3.6.x (32-bit) Setup pop-up window will appear.

 **Note:** Ensure to select the checkbox in the pop up
     - Add Python 3.6.x to PATH
     - Install Launcher for all users

- Click on the **Install Now** message.

 **Note:** Administrator permission is needed to continue the installation.

- Click the Yes button when asked for Administrator permission.

Python has been successfully installed in your Windows machine.

3.Verifying

To try to verify installation,

    Go to Start Menu (All Programs) in the home screen.
    Search Python.
    It will open the Python command line prompt terminal.

Jupyter Notebook Installation

Install Jupyter Notebook in Ubuntu using Python3

Prerequisite : Python3 must be installed in Ubuntu

1.Open Terminal

test@domain.com:$ cd
test@domain.com:$ mkdir ~/jupyterNotebook
test@domain.com:$ cd jupyterNotebook
test@domain.com:~/jupyterNotebook$ virtualenv -p python3 env
test@domain.com:~/jupyterNotebook $ source env/bin/activate
(env) test@domain.com:~/jupyterNotebook$ pip install jupyter

2.Run Jupyter Notebook from Ubuntu

Open Terminal

test@domain.com:$ cd jupyterNotebook
test@domain.com:~/jupyterNotebook $ source env/bin/activate
(env) test@domain.com:~/jupyterNotebook$ jupyter notebook

Install Jupyter Notebook in Windows

1.Install Python3 and create a virtual environment in command prompt.

C:\>C:\Python35\python -m venv c:\path\to\myenv

2.Activate the Virtual environment

C:\> <venv>\Scripts\activate.bat

3.Install Jupyter Notebook

C:\> pip install jupyter

4.Start Jupyter Notebook

C:\> jupyter notebook

This will start the jupyter notebook in windows.
