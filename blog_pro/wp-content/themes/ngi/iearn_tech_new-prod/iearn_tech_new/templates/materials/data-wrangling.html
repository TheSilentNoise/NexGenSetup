Introduction

Definition

Data Wrangling is the process of transforming raw data in an appropriate form which can be consumed by downstream applications like Data Analytics.

In a typical data analytics project data wrangling occupies a significant proportion of total effort.

Pandas is a python library which is extensively used for Data Wrangling

Stacking

Stacking helps to pivot a level of column labels

Very useful to represent hierarchical relationships

It returns a dataframe having hierarchical index with a new inner-most level of row labels
