Course Description :

About the course

This course is designed to torture the data for it to confess. The course will introduce learner to various techniques to organize and analyze data. The learner will be introduced to various munging, wrangling and engineering and analysis techniques. At the end of the course will be able to leverage data analysis skills to undertake data driven decisions.

Who should go for this course

The course is ideally suited for any one who is interested to work as Data Analyst and need to take data driven decisions. It is also advisable for learners who want to pursue career in the field of data analytics, data engineering, machine learning and AI.

Prerequisites

The learner should have interest in learning about data analysis and should have ideally completed Fundamentals and Data Structure Course.
